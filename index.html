## Vue.js ![Sample image](images/Vue.PNG) --- go --- ## Vue.js是什麼? 簡單小巧的核心，漸進式技術堆疊，足以應付各種規模應用 [Vue Link](https://vuejs.org/) --- * 可重複使用的元件。 * 前端路由。 * 狀態管理。 * 虛擬DOM。 --- ## MVVM模式 當View(視圖層)產生變化時，會自動更新到ViewModel(視圖模型)。View和ViewModel之間透過雙向綁定 (data-binding)建立聯繫 --- ![MVVM image](images/MVVM.PNG) --- ## View層 --- ![View dome image](images/View.PNG) --- ## Model層 --- ```javascript let data = { title: '', list: [] } ``` --- ## ViewModel層 --- ```javascript let vm = new Vue({ el: '#app', data: data, methods: { add() { this.list.push(this.title) this.title = '' } } }) ``` --- ## Vue.js 有什麼不同 --- ## Jquery + 操作DOM + 綁定事件 --- ![Jquery image](images/Jquery.PNG) --- ## Vue.js 改寫 視圖與資料拆開 --- ![Jquery image](images/Vdemo.PNG) --- ## 如何使用Vue.js --- 1. 可透過Script 載入CDN 2. CLI [安裝教學](https://cn.vuejs.org/v2/guide/installation.html) --- [第一個Vue應用Hello World?](https://jsbin.com/horazigafe/edit?html,js,output) --- 建置函數Vue可以建置Vue的根實例並啟動Vue應用 ```javascript var app = new Vue({ //選項 }) ``` el用於指定一個頁面中已存在的DOM元素來掛載Vue,可以是HTMLElement,也可以是CSS選擇器 ```javascript var app = new Vue({ el: document.getElementById('app') // or '#app' }) ``` 掛載成功後可以使用 app.$el 來取得元素 --- 那如何取得data物件裡的屬性呢? --- ```javascript var app = new Vue({ el: '#app', data: { a:2 } }) console.log(app.a); // 2 ``` --- ## 先介紹兩個常用的生命週期 --- created 實例建立完成後呼叫(用於初始化資料) --- mounted el掛載到實例上後呼叫 --- ```javascript var app = new Vue({ el: '#app', data: { a:2 }, created() { console.log(this.a); // 2 }, mounted() { console.log(this.$el); }, }) ``` --- 內插運算式 --- [計時器](https://jsbin.com/lusibediwo/edit?html,js,output) --- 今天就先這樣吧....期待下次再見!
